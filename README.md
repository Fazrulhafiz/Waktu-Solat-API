# Waktu-Solat-jQuery
Fetch and display Malaysia's Prayer Time dynamically using jQuery and Solat.io API
- Original Project : https://github.com/afzafri/Waktu-Solat
- This version does not use PHP. Just HTML, Javascript and jQuery
